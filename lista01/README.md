# POO_Python

Este repositório contém a resolução da **Lista de Exercícios de Programação Orientada a Objetos (POO) em Python**, desenvolvida como atividade prática da disciplina. Os exercícios têm como objetivo consolidar os principais conceitos da programação orientada a objetos por meio da implementação de soluções para problemas práticos.

## 📚 Objetivos

- Praticar os fundamentos da Programação Orientada a Objetos (POO).
- Aplicar conceitos como encapsulamento, herança, polimorfismo, composição, agregação e dependência.
- Desenvolver código organizado, reutilizável e de fácil manutenção.
- Utilizar tratamento de exceções para validação de entradas quando necessário.

---

## 📋 Exercícios

A lista é composta por **12 exercícios**, contemplando os seguintes conteúdos:

| Arquivo | Descrição |
|---------|-----------|
| **Ex1.py** | Progressão Aritmética utilizando orientação a objetos. |
| **Ex2.py** | Manipulação de Strings. |
| **Ex3.py** | Criptografia de texto por substituição de vogais. |
| **Ex4.py** | Sistema de Carrinho de Compras com encapsulamento. |
| **Ex5.py** | Dependência entre Documento e Impressora. |
| **Ex6.py** | Agregação entre Departamento e Funcionário. |
| **Ex7.py** | Composição entre Casa e Cômodos. |
| **Ex8.py** | Sistema de RPG utilizando herança e polimorfismo. |
| **Ex9.py** | Plataforma de Cursos Online utilizando herança e polimorfismo. |
| **Ex10.py** | Sistema dos Cavaleiros do Zodíaco utilizando herança múltipla. |
| **Ex11.py** | Sistema de Transporte Público com classes abstratas e tratamento de exceções. |
| **Ex12.py** | Torneio Dragon Ball utilizando abstração total, herança e polimorfismo. |

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/brnaum/POO_Python.git
```

2. Entre na pasta do projeto:

```bash
cd POO_Python
```

3. Execute o exercício desejado:

```bash
python Ex1.py
```

ou

```bash
python3 Ex1.py
```
