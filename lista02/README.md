# Python_POO - Lista de Exercícios 2

Este repositório contém as soluções da **Lista de Exercícios 2** da disciplina de **Programação Orientada a Objetos (POO)**, desenvolvidas em **Python**.

## 📋 Sobre a lista

A lista tem como objetivo aprofundar os conceitos de Programação Orientada a Objetos por meio da implementação de sistemas completos utilizando classes, herança, polimorfismo, encapsulamento, abstração, tratamento de exceções e menus interativos.

Os exercícios abordam diferentes cenários práticos, como:

- Sistema de gerenciamento de cursos e alunos;
- Sistema bancário com contas, clientes, Pix e histórico de titularidade;
- Sistema de empréstimos e reservas de recursos para uma instituição de ensino;
- Sistema de votação eletrônica com anonimização de dados e persistência em JSON.

## 📁 Estrutura do repositório

Os arquivos estão organizados da seguinte forma:

```
POO_Python/
│
├── Ex1.py
├── Ex2.py
├── Ex3.py
├── Ex4.py
└── README.md
```

Cada arquivo corresponde a um exercício da lista.

## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/Python_POO.git
```

2. Acesse a pasta do projeto:

```bash
cd Python_POO
```

3. Execute o exercício desejado:

```bash
python Ex1.py
```

(Substitua `Ex1.py` pelo arquivo que deseja executar.)
